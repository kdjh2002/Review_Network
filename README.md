# Review_Network

본 평가는 언리얼 엔진의 네트워크 기능을 활용한 멀티플레이 게임 로직 및 UI 동기화 구현 능력을 평가하기 위한 과제이다. 학생은 아래 요구 사항을 충족하는 프로젝트를 구현하고 결과물을 제출해야 한다.

# 1. 프로젝트 기본 구성  

1. 언리얼 엔진의 3인칭 템플릿(Third Person Template) 을 기반으로 프로젝트를 구성할 것.  

# 2. NetGameMode 구현하기

1. 본 게임은 1:1 멀티플레이를 기준으로 진행한다.  
2. 한 게임의 진행 시간은 1분으로 설정한다.  
3. 게임에서 승리하기 위해서는 상대보다 더 많은 픽업 아이템을 획득하여야 한다.  


# 3. NetSpawner 구현하기

1. 게임이 진행되는 동안, 3초 간격으로 픽업 아이템이 맵 내 랜덤한 위치에 드랍되도록 구현해야 한다.  

# 4. NetHUD 구현하기

1. 게임 진행 중 남은 게임 시간을 HUD를 통해 실시간으로 확인할 수 있어야 한다.  
2. 자신과 상대가 획득한 픽업 아이템의 개수를 HUD를 통해 실시간으로 확인할 수 있어야 한다.  
3. 위 HUD 요소들은 네트워크 환경에서 각 클라이언트에 정확하게 동기화되어 표시되어야 한다.  
4. 게임 시간이 종료되면 승리 및 패배를 판정하여, 각 플레이어에게 승리 또는 패배 결과가 명확하게 UI로 표시되어야 한다.
